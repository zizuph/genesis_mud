Cheiropoto (Elemental At Will Damage)
=====================================
Guild type          : Layman
Effect              : Elemental Damage spell that uses random elements to
                      attack foes. It is an at will spell, meaning that the
                      mana cost is variable depending on player size, and
                      is usually at the same rate as regen rate.
                      
                      For more information, see description in 
                      /d/Calia/guilds/academics/help/spells/cheiropoto

Peaceful            : No

Spell Power         : Power 3 - Intermediate
Area of effect      : One Enemy Target
Duration            : Instant
